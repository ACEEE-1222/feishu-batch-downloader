📖 配置凭证完整流程（点击展开/收起）
步骤 1：创建应用
访问 飞书开放平台 → 点击"创建企业自建应用"

步骤 2：获取凭证
进入"凭证与基础信息"页面，分别复制：
• App ID（以 cli_ 开头）
• App Secret（长字符串，与 App ID 不同）

步骤 3：配置权限（重要！）
进入"权限管理"页面，点击"批量配置权限"，粘贴以下 JSON：
{
  "scopes": {
    "tenant": [
      "docx:document:readonly",
      "drive:drive:readonly",
      "wiki:wiki:readonly"
    ],
    "user": [
      "docx:document:readonly",
      "wiki:wiki:readonly"
    ]
  }
}
或手动添加以下权限：
• docx:document:readonly - 读取文档内容
• drive:drive:readonly - 读取云空间文件
• wiki:wiki:readonly - 读取知识库内容

步骤 4：发布应用
进入"版本管理与发布" → 创建版本 → 申请发布 → 发布到企业内部

步骤 5：保存配置
在下方输入框填入 App ID 和 App Secret，点击"保存配置"
📄 下载文档
粘贴飞书文档链接或文档 ID，支持批量下载（每行一个）
输入完整路径（如 /Users/你的用户名/Documents/feishu）或使用默认下载文件夹
选择文档导出格式
